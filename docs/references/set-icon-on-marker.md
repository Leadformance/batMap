## setIconOnMarker

> setIconOnMarker(marker, iconType, isLabeled = true)

Set a given icon type to a marker.

### Parameters

#### `marker` - String|Object

The marker object or its id.

#### `iconType` - String

The marker icon type to set.

#### `isLabeled` - Boolean

Enable/disable label on marker.
